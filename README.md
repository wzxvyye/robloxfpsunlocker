Usage
Download the latest release from https://github.com/wzxvyye/robloxfpsunlocker/releases
Extract robloxfpsunlocker.zip into a folder
Run robloxfpsunlocker.exe before or after starting Roblox
After selecting FPS, don't forget to restart

FAQ

How can I see my FPS?
Press Shift+F5 in-game to view your FPS.

Does this work for Mac?
No. Roblox FPS Unlocker was written only for the Windows platform.

Why do I get a "Failed to connect to Github" error?
This error means Roblox FPS Unlocker could not connect to the Internet to check for updates. This may be due to your anti-virus, computer firewall, network firewall, or etc. blocking the request. The error can be safely ignored by pressing "Ok".

How do I uninstall Roblox FPS Unlocker?
Roblox FPS Unlocker does not install itself anywhere. It can be deleted by simply exiting the program if it is open (tray icon->Exit) and deleting robloxfpsunlocker.exe.
